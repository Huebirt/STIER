# STier – Tier List Maker

A simple, browser-based tier list maker. Search for Steam games, upload your own images, drag and drop them into customizable tier rows, and export the result as a PNG.

🔗 **Live site:** [mcdevie.github.io/STIER](https://huebirt.github.io/STIER/)

## Features

- 🔍 **Steam Game Search** – Search the Steam store and add game images directly
- 📤 **Image Upload** – Upload individual images or entire folders
- 🖱️ **Drag & Drop** – Drag images between the game pool and tier rows, reorder freely
- ✏️ **Editable Labels** – Click any tier label to rename it
- 🎨 **Custom Colors** – Right-click a tier label to change its color
- ➕ **Add / Delete Rows** – Add new tiers or remove existing ones
- 📸 **Export as PNG** – Download your tier list as an image

## Tech Stack

- HTML / CSS / JavaScript (vanilla, no frameworks)
- [html2canvas](https://html2canvas.hertzen.com/) for PNG export
- Steam Store Search API (via CORS proxy)

## Getting Started

No build step required — just open `index.html` in a browser, or visit the live site above.

To run locally:

```
git clone https://github.com/McDevie/STIER.git
cd STIER
open index.html
```
